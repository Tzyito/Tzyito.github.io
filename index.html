Posted on 2019-03-23 | Post modified: 2019-03-23 | In 教程 , Vue开发

最近由于在学习vue的过程中，遇到过很多问题（例如：项目的搭建，运行整个项目，debug，配置文件等）所以想着为虾米不把遇到的问题记下来，然后以后遇到同样的问题也可以参考一下 \^/hhh!!

何谓Vue？

vue是一个前端开发的框架

​ vue是由前谷歌员工尤雨溪及其团队开发的一个js框架，同样类似的框架还有react，Angular. （跳过介绍其他框架的环节）

`vue`现在有良好的中文文档 [点击这里](https://cn.vuejs.org/v2/guide/)，你可以通过文档了解到更加详细的有关于vue的知识。

如何使用Vue？

安装Vue

直接用<script>引入

直接下载并用<script>标签引入，Vue会被注册为一个全局变量

对于简单的学习，可以这样使用最新版本

NPM

在用Vue构建大型应用时使用npm安装

命令行行工具（CLI）

Vue提供了一个官方的CLI，为单页面应用(SPA)快速搭建繁琐的脚手架。(相信我，没有学习webpack前的项目搭建是十分的痛苦的！！┭┮﹏┭┮) Vue cli之后会是我们经常使用脚手架工具。更多详情请查阅Vue-cli官方文档

CLI 工具假定用户对 Node.js 和相关构建工具有一定程度的了解。如果你是新手，我们强烈建议先在不用构建工具的情况下通读指南，在熟悉 Vue 本身之后再使用 CLI。

项目的入口

整个项目的入口文件是main.js，然后所有.vue文件最后都是通过这个文件挂载到网页上进行渲染，而App.vue文件相当于我们写的Html的body区域。由于Vue使用的是组件化思想，所以每个部分都被划分成了一个小的组件，然后再通过导出组件并在App.vue里注册并使用。具体如何体现，我将在以后的学习中说明

其他文件说明

src文件夹：
assets放的是项目的style,imgage等文件
